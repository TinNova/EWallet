# EWallet - Android App

### App Functionality
Check your crypto account balances. The first screen will show a hardcoded wallet address, the second screen will show a search bar where you can see your ERC-20 token balances.

## Tech-Stack

* Kotlin
* Dagger Hily
* Coroutines
* Compose
* Architecture
  * Clean Architecture
  * MVI
* Jetpack Navigation 
* Testing
  * JUnit5
  * Mockk
